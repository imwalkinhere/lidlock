# LidLock 1.1.0 (superseded)

This build was superseded by 1.1.1 after a helper validation bug was found. The
public release asset was removed; use 1.1.1 or newer.

Password-free lid control release for LidLock on macOS.

## Download

The attached `LidLock.dmg` is signed with Apple Developer ID and notarized by Apple.

Homebrew:

```sh
brew install --cask DelPage/lidlock/lidlock
```

SHA-256:

```text
09c2a796d81e1fcb8930dd30f356aca8a41413aadf66277b4f14186071d0c1b7  LidLock.dmg
```

## What Changed

- Added an optional signed privileged helper in Settings so Survive Lid Close
  can turn on and off without asking for the admin password every time.
- Added a warning dialog before installing the helper.
- Kept the normal macOS admin prompt as the fallback when the helper is off.
- Added release checks that fail if the helper or launchd plist is missing from
  the signed app.

## Requirements

- macOS 13 Ventura or newer.
- A MacBook for lid-close workflows. Other Macs can still use keep-awake and
  display controls.

## Privacy

LidLock has no accounts, analytics, telemetry, advertising, or automatic network
access during normal operation. The optional helper is local to your Mac and is
used only for LidLock's lid-close sleep setting.

Privacy policy: https://delpageinc.com/privacy/lidlock
